# Формулы нумерологии (v1)

## Число жизненного пути
Сумма всех цифр даты рождения (ДДММГГГГ) с редукцией до 1–9.
Мастер-числа 11 и 22 сохраняются.

## Личное число года
День + месяц рождения + текущий год → редукция.

## Личное число месяца
День + месяц рождения + месяц + год календаря → редукция.

## Число дня
День + месяц рождения + текущая дата → редукция.

## Число богатства (упрощённое v1)
`день×2 + месяц×3 + (год % 100)` → редукция.

## Совместимость
Сумма чисел жизненного пути двух дат → редукция.

> Точные формулы метода Хшановской (Матрица Судьбы) уточняются у заказчика.
