# Админ-команды (Telegram)

## /confirm_payment \<id\>
Подтверждает ручную оплату и выдаёт полный разбор или активирует пакет.

## /stats
Количество зарегистрированных пользователей.

## /llm_cost
Суммарный расход токенов Gemini 2.5 Pro: всего и за сегодня, в рублях.

Каждый LLM-запрос логируется в `llm_usage_logs` и в structlog (`llm_token_usage`).

# Веб-дэшборд

Отдельная панель для заявок на оплату и истории переписки.

## Запуск

1. В `.env` задай пароль:
   ```
   DASHBOARD_PASSWORD=your_secure_password
   ```
2. Запусти (в отдельном терминале, параллельно с ботом):
   ```bash
   ./scripts/run_dashboard.sh
   ```
3. Открой в браузере: http://127.0.0.1:8080

## Разделы

- **Заявки** — pending / paid / cancelled, кнопки «Подтвердить» и «Отклонить»
- **Пользователи** — поиск по имени, username или Telegram ID
- **Переписка** — история сообщений пользователя ↔ бот (входящие и исходящие)

История пишется в таблицу `chat_messages` после перезапуска бота с новой версией.
